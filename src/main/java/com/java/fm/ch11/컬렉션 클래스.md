ArrayList, Vector -> Stack (LIFO)
                  -> LinkedList (추가, 삭제기능 향상)
                  -> HashMap(New), Hashtable(old) (검색기능 향상)
                  
LinkedList -> Queue (FIFO)
           -> TreeMap (검색, 범위검색, 정렬기능 향상) -> TreeSet
           -> HashMap(New), Hashtable(old) (검색기능 향상)
           
HashMap(New), Hashtable(old)   -> Properties (String(key), String(value))
 (Object(key), Object(value)   -> LinkedHashMap (순서유지기능 향상)
                               -> HashSet -> LinkedHashSet (순서유지 기능 향상)
